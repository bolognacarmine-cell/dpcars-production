# DP CARS
Sito web ufficiale DP CARS – Auto e Moto di Qualità a Marcianise.

## Struttura del progetto
- **Frontend**: sito statico (HTML, CSS, JavaScript + Vue via CDN)  
- **Backend**: API Node.js (separata, non inclusa nel frontend statico)

## Deploy attuale
**Tutto il progetto è deployato e gestito su Render**[](https://render.com).

- **Frontend** → https://dpcars-production.onrender.com  
  - Deploy automatico su push nel branch principale  
  - Ambiente: Static Site (o Web Service se serve Node per build)  
  - Nessuna dipendenza da Netlify

**Nota importante**: il frontend era inizialmente ospitato su Netlify, ma è stato completamente migrato su Render.  
Non ci sono più configurazioni, file, variabili d'ambiente o dipendenze legate a Netlify nel repository o nel sito live.
